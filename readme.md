**doc-to-pdf**

> This package generate PDF files from documents.

*Tested:*
 - doc
 - docx
 - ppt
 - pptx
 - pdf
 - xls
 - rtfodt
 - odp


**Install**

```
yarn add doc-to-pdf
```

**Usage**

```javascript
const P = require('doc-to-pdf')

(async start() {
  await P('./demo.pptx')
})()
```
